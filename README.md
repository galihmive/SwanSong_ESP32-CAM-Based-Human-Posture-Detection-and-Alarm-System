# SwanSong-ESP32-CAM-Human-Posture-Alarm
Proyek ini merupakan sistem berbasis IoT yang dirancang untuk mendeteksi pola tubuh manusia dalam kondisi tidur dengan posisi berdiri menggunakan ESP32-CAM, kemudian memberikan respons berupa alarm suara dan getaran melalui modul terpisah. Sistem ini dibagi menjadi dua bagian utama: unit deteksi visual dan unit output peringatan.
